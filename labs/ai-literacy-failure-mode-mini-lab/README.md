# AI Literacy Failure-Mode Mini-Lab

A 20-minute synthetic exercise for practicing verification of AI answers.

## Learning goal

Learners practice spotting four common AI-answer states:

1. accurate and supported,
2. plausible but wrong,
3. incomplete,
4. unsupported or overconfident.

The point is not to shame AI use. The point is to build the habit of checking the answer against the task, the source, and the decision risk.

## Time box

20 minutes.

## Materials

- `worksheet.md`
- `answer-key.md`

## How to run

1. Give learners the scenario and three AI answers in `worksheet.md`.
2. Before checking, ask them to score confidence for each answer from 1-5.
3. Let them inspect the evidence box and mark each answer as accurate, plausible-wrong, incomplete, or unsupported.
4. Ask for a revised confidence score.
5. Compare with `answer-key.md`.
6. Discuss: what changed your confidence, and what would you check before acting?

## Safety boundary

This lab uses synthetic, non-high-stakes material. Do not use it as legal, medical, financial, or safety advice.
