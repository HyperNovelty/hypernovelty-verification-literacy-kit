# Security Policy

## Scope

This repository is an educational kit with synthetic examples and Markdown templates. It should not store secrets, credentials, private traces, private source material, account workflows, or production logs.

## Reporting a problem

If you find a security issue in this repository, please open a GitHub issue with a minimal description that does not disclose secrets or private data.

If the issue involves a leaked secret or private material, do not repost the secret in an issue. Remove the material from your fork/clone and contact the maintainer through the repository owner profile.

## Maintainer rules

- Do not commit credentials, cookies, API keys, private logs, or account screenshots.
- Use synthetic examples by default.
- Strip private paths and personal data before publishing examples.
- Do not use this repo to automate public posting, payments, account changes, trading, or credential operations.
