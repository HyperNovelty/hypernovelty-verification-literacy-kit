# Code of Conduct

This project is for practical, public-safe tools and examples around AI-era adaptation, verification, source trust, and human review.

Be respectful, concrete, and constructive. Do not use issues or pull requests to post secrets, credentials, private data, customer records, student records, medical/legal/financial claims, harassment, spam, or unsafe automation instructions.

Maintainers may close or edit contributions that violate these boundaries or that make unsupported claims about what the project can do.
